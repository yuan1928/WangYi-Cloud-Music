# 华科-腾讯web前端开发菁英课程大作业：仿照网易云音乐设计的音乐APP
## (1)项目相关
### 1)安装依赖：`npm install`
### 2)运行项目：`npm start`
### 3)构建项目：`npm run build`
## (2)技术栈
### 1)开发语言：TypeScript
### 2)开发框架：React
### 3)通信协议：https
### 4)路由管理库：react-router-dom
### 5)UI组件库：Ant Design
### 6)请求库：Axios
